summary: in this tutorial you will learn about diffreance between map, filter, forEach, find
map()

Map function received a function as parameter and this function not
modify original array return new array

let numbers = [1,2,3,4]
console.log(numbers.map(number => number % 2 === 0))
output : [false,true,false,true]
forEach()

ForEach function received a function as a parameter and forEach
function
not modify this array and forEach function not return this is always undefined

let numbers = [1,2,3,4,5]
console.log(numbers.forEach(number => number % 2 === 0))
output: undefiend
filter()

filter function received a function as a parameter if not match a any
data filter function return empty array elsif match anything data
function return this result inner array

let numbers = [1,2,3,4,5]
console.log(numbers.filter(number => number % 2 === 0))
output : [2,4]
find()

javascript find function argument another function this return only first result match

let numbers = [1,2,3,4,5]
console.log(numbers.map(number => number%2 === 0))
output : 2
